🏛️ Church Calendar LINE Bot

✅ Server is running successfully!

Available API Endpoints:

GET /api/activities - View all activities
POST /api/activities - Create new activity (Admin only)
PUT /api/activities - Update activity (Admin only)
DELETE /api/activities - Delete activity (Admin only)
POST /api/reminders - Send reminders (Cron jobs)
POST /api/webhook - LINE Bot webhook

LINE Bot Commands:

  • help - Show help message
  • 查看 全部 - View all activities
  • 查看 這個月 - View current month activities
  • 查看 這個禮拜 - View current week activities
  • 查看 [ID] - View specific activity

This is a serverless function project deployed on Vercel.
